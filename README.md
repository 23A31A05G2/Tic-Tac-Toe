# Tic Tac Toe Game ⭕❌

## Description

A Java console-based Tic Tac Toe game for two players. Players take turns marking spaces on a 3×3 grid, and the program checks for winning combinations or a draw.

## Features

* Two-player gameplay
* 3×3 game board
* Win detection
* Draw detection
* User-friendly interface

## Technologies Used

* Java
* Arrays
* Loops
* Conditional Statements

## How to Run

1. Compile the program:
   javac Main.java

2. Run the program:
   java Main

## Learning Outcomes

* Array Manipulation
* Game Logic Design
* Problem Solving
* User Interaction
